⚠️ ¿Windows muestra una advertencia al abrir el archivo?

Es normal. Windows SmartScreen muestra esa alerta a todas las aplicaciones nuevas que aún no tienen "reputación" oficial — pero ✓ Microsoft Defender ya verificó SubFlow.exe y confirmó que está limpio (sin malware). Solo le falta acumular descargas para que la advertencia desaparezca automáticamente.

Para abrirlo:

  1. Doble clic en SubFlow.exe
  2. Si aparece "Windows protegió tu PC", haz clic en "Más información"
  3. Luego clic en "Ejecutar de todas formas"
